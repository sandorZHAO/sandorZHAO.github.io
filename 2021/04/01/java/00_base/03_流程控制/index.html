块作用域

块是指一堆花括号括起来的若干条的Java语句。
块确定了变量的作用域。
一个块可以嵌套在另一个块中。
不能再嵌套着的两个块中声明同名的变量。

break

break语句可以跳出当前循环；
break语句通常配合if，在满足条件时提前结束整个循环；
break语句总是跳出最近的一层循环；
break可以中断switch多重选择依次向下执行；

continue

continue语句可以提前结束本次循环；
continue语句通常配合if，在满足条件时提前结束本次循环。
